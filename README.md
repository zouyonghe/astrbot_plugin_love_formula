# 💖 Love Formula: 赛博恋爱演化诊断插件

> **“假如这个群是你的恋人，你今天的表现是顶级偶像还是败犬 NPC？”**

本插件基于“恋爱动力学”逻辑，通过分析群成员每日的社交行为数据，生成一份极具“二次元/亚文化”色彩的恋爱成分诊断报告。

## 🎨 视觉风格 (Galgame UI)
提供精致的报表渲染，包含：
- **群聊好感度**: 综合评价你今日的社交能量。
- **动态指标网格**: 实时展示“纯爱值”、“存在感”与“败犬值”。
- **恋爱演化算式**: 使用 LaTeX 渲染的数学逻辑展示。
- **AI 裁判点评**: 毒舌、中二且精准的亚文化行为诊断。

---

## 📊 核心指标映射 (Terminology)

我们抛弃了枯燥的统计学术语，全面拥抱亚文化梗语境：

| 维度 | ACG/潮流称呼 | **💕 赛博恋爱映射** | **📈 计算逻辑** |
| :--- | :--- | :--- | :--- |
| **Simp ($E$)** | **纯爱值** | **付出与诚意**：指主动发起的交互和投入成本。 | 发言数、戳一戳、文字长度。 |
| **Vibe ($V$)** | **存在感** | **吸引力与光环**：代表他人对你的正向反馈。 | 被回复数、被贴贴/回应数。 |
| **Loser ($B$)** | **败犬值** | **尴尬感与劝退力**：社交失误或氛围破坏。 | **撤回消息**、尬聊等行为。 |
| **Active** | **营业频率** | 今日活跃度，像偶像一样营业。 | 每日发言频次。 |
| **Power** | **小作文功率** | 每条消息的“情感厚度”。 | 平均每条消息的字数。 |

---

## 🎭 角色人设 (Archetypes)

根据各项指标的权重均衡，你会被判定为以下几种人设：
- **纯爱战神 (反讽)**: 高投入、低回报的深情（卑微）代表。
- **现充达人**: 极低投入却随处散发魅力的社交掌控者。
- **傲娇败犬**: 魅力虽高，但因频繁撤回或犹豫不决导致“败犬气息”浓郁。
- **顶级偶像**: 群内的白月光，几乎不主动但深受宠爱。
- **背景板 NPC**: 处于潜水边缘，社交存在感极低的过路人。

## 🤖 AI 行为诊断
插件通过调用 LLM，化身为 **恋爱法庭的恶役大小姐/审判官**，对你的数据进行辛辣点评：
- *“哼，纯度94%的纯爱圣骑士？可惜存在感稀薄如幽灵，败犬气息已悄然弥漫。”*
- *“被贴贴7次伴随2次撤回，揭露了残酷真相：那些贴贴不过是群友随手施舍的‘怜悯型营业’。”*

---

## 🛠️ 安装与配置
1. 在 AstrBot 插件市场安装 `astrbot_plugin_love_formula`。
2. 配置项：
   - `enable_llm_commentary`: 是否开启 AI 点评 (默认开启)。
   - `min_msg_threshold`: 触发诊断的最小发言数 (默认 3 条)。
   - `theme`: 默认为 `galgame`。
